Getting Started

ExecTrace is a real-time C++ profiling platform. Get started in 3 steps:

1. Create an Account

Sign up at /register with your email and password.

2. Create a Project

Login and create a new project in the workspace. Copy your Project API Key

3. Integrate the SDK

Download the SDK and add it to your C++ project:

#include "exectrace.hpp"

int main() {
    ExecTrace::init("your_project_key", "v1.0.0");
    TRACE_FUNCTION();
    
    return 0;
}

API Reference

init()

Initialize the SDK with your project API key.

ExecTrace::init(project_key, app_version, host, port);

TRACE_FUNCTION()

Automatically track function execution time and memory usage.

void myFunction() {
    TRACE_FUNCTION();
}

Dashboard

View your profiling data in real-time at /dashboard?key=YOUR_PROJECT_KEY
