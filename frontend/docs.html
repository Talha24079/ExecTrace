Quick Start

Include the SDK and use the macro.

#include "exectrace.hpp"

int main() {
    ExecTrace::init("YOUR_API_KEY");
    
    TRACE_FUNCTION(); // Auto-logs this function
    
    return 0;
}

API Reference

Function Description
init(key) Connect to localhost:8080
TRACE_FUNCTION() Profile current function
